\textbf{5．極座標と極方程式}

直交座標系$O-xy$に関する点$P$の直交座標$(x,y)$と
\textbf{極座標}$(r,\theta)$との関係は，
\begin{equation}
	x	=	r	\cos	\theta,	\quad	y=r	\sin	\theta
											\quad	(r	>	0;	\text{図\ref{fig20.8}})
\end{equation}
これから逆に，
\begin{figure}[hbtp]
	\centering
	\includegraphics[bb=0 0 800 600,clip,width=7.5cm,height=5cm]{fig20x8.bmp}
	\caption{極座標}
	\label{fig20.8}
\end{figure}
\begin{equation}
	x	=	r	=	\sqrt{x^2+y^2},	\tan	\theta	=	\frac{y}{x}.
\end{equation}
極座標で表された曲線の方程式を\textbf{極方程式}という．

\例{4}
Cardioid（ハート型）：$r=\alpha	(1+	\cos	\theta)	\quad	(\alpha	>	0).$

$r	=	a\{	1+	\cos	(-\theta)\}=a(1+	\cos	\theta)$であるから，
グラフは$x-$軸に関して対称である．また，
$r=a(1	+	\cos	\theta	)	\leqq	2a$であるから，曲線は中心$(0,0)$，半径$2a$の円内にある．
\[
	\frac{dr}{d\theta}	=	-	a	\sin	\theta	\leqq	0	\quad	(0	\leqq	\theta	\leqq	\pi)
\]
$\theta	=	0,\pi$で$\displaystyle	\frac{dr}{d\theta}=0$．

$y$座標に関しては，
\begin{align*}
	y							&=	r	\sin	\theta	=	a	(1	+	\cos	\theta	)	\sin	\theta	\\
	\frac{dy}{dr}	&=	a	(	2	\cos	\theta	-1)(\cos	\theta	+1)
\end{align*}
$\displaystyle	\frac{dy}{dr}$を解いて
$\cos	\theta	=	-1$，このとき$\displaystyle	\theta	=	\frac{3\pi}{2}$（不適）．
$\displaystyle	\cos	\theta	=	-\frac{1}{2}$，このとき$\displaystyle	\theta	=	\frac{\pi}{3}$，
$\displaystyle	y=\frac{3\sqrt{3}a}{4}$．

$x$座標に関しては，
\begin{align*}
	x							&=	r	\cos	\theta	=	a	(1	+	\cos	\theta	)	\cos	\theta	\\
	\frac{dx}{dr}	&=	-a	\sin	\theta	(	2	\cos	\theta	+1)
\end{align*}
$\displaystyle	\frac{dx}{dr}$を解いて
$\sin	\theta	=	0$，このとき$\displaystyle	\theta	=	0,\pi$．
$\displaystyle	\cos	\theta	=	-\frac{1}{2}$，このとき$\displaystyle	\theta	=	\frac{2\pi}{3}$，
$\displaystyle	r=\frac{1}{2}a$，$\displaystyle	x=-\frac{1}{4}a$


したがって，$r$は$0	\leqq	\theta	\leqq	\pi$で減少の状態であり(表\ref{tab20.4})，
グラフの概形は，図\ref{fig20.9}のようになる．
