すなわち，
\begin{equation*}
	y=\sin^{-1}	x.	\Longleftrightarrow	
	x=\sin	y	
	\quad	\left(	-\frac{\pi}{2}	\leqq	y	\leqq	\frac{\pi}{2}	\right).
	\quad	\text{\textbf{($\sin^{-1}	x$の定義)}}
\end{equation*}
\begin{equation}
	(\sin^{-1}	x)'	=	\frac{1}{\sqrt{1-x^2}}	\quad\quad	(-1	<	x	<	1).
	\label{eq10.1}
\end{equation}
\証明
逆関数の微分の公式によって，
\[
	\frac{d	\sin^{-1}	x}{dx}	
		=	\frac{1}{\displaystyle	\frac{d	\sin	y}{dy}}
		=	\frac{1}{\cos	y}	=	\text{＊}.													
\]
ここで，$y=\sin^{-1}	x	$は主値：
$\displaystyle	-\frac{\pi}{2}	\leqq	y	\leqq	\frac{\pi}{2}$をとるから，
この範囲で$\cos	y	\geqq	-$であることに注意すれば，
\[
	\text{＊}	=	\frac{1}{\sqrt{1-\sin^2	y}}
						=	\frac{1}{1-x^2}	\quad\quad	(-1	<	x	<	1).
\]
(証明終)
