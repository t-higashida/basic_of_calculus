{\Huge \textbf{まえがき}}
\vspace{1cm}

この本は，新しく大学に入学したばかりの教養課程の学生たちを対象にして，
彼らが，微分積分学を，出来るだけわかりやすく，興味をもって学習することができるように
書かれた教科書である．

高校の数学の教科書と大学の微分積分学の教科書を比べてみると，一般的に
いって，この隔たりが，学ぶものの数学に対する興味を失わせる一つの要因と
考えられる．この本では，出来るだけそのような隔たりをなくして，
あまり抵抗を感ずることなく，高校での数学教育の延長として，円滑に，微分積分学を
学ぶことができるように心を配った．

昭和58年1月に，初版を発行させていただいて以来，永い間，多くの方々に，ご使用していただき，
色々と有益なご指摘，ご助言を頂いた．
そのようなご指摘，ご助言を参考にしながら，今回，改訂させていただくことになった．
わかりやすく，興味をもって学習することができるという初版の精神を尊重しながら，
初版より，より一層やさしく理解できるような内容になるように努めた．

この本は，高校の「数学Ⅰ」の範囲を予備知識として，理解することができるように書かれている．
したがって，高校での「微分・積分」の過程を終えた人たちを対象に教える場合には，
この本のある部分は省略して，先に進むことができる．実際，著者もそのように実行しており，
この本もそのことを心に留めながら書いた．どこを省略するかは，教える人の判断に委ねたいが，
１つの，案としては，次の各節を省略する方法がある：
\begin{list}{}{}
	\item	\S	1.	極限値
	\item	\S	2.	連続関数
	\item	\S	3.　導関数
	\item	\S	4.	微分法の公式（4項の「パラメター表示された関数の微分」は教える）
	\item	\S	5.	べきの微分
	\item	\S	6.	三角関数の微分
	\item	\S	7.	指数関数と対数関数の微分
	\item	\S	20.	関数の増減と極値（定理20.3および5項の「極座標と極方程式」は教える）
	\item	\S	40.	定積分と不定積分
	\item	\S	41.	定積分の計算（例６は教える）
	\item	\S	42.	平面図形の面積（2項，3項は教える）
	\item	\S	58.	無限数列
	\item	\S	59.	無限級数
\end{list}
省略することによって，本書の講義に要する総時間数を，かなり短縮することができるであろう．

各節（\S	1,	\S	2,など）ごとにあげた問は，本文と直接関係のある基礎的ないしは基本的なものを
採用し，本文の内容の理解を深めるように努めた．なお，\textbf{問題}については，
『微分積分学問題集（改訂版）』（培風館発行）
が，この本と平行して出版される予定である．この問題集には，この本の説（\S	1,	\S	2,
など）の分類に基づいて，問題を，
\begin{list}{}{}
	\item	A:	やさしい基礎的な問題，
	\item	B:	基本的で標準的な問題，
	\item	C:	やや高度な問題および応用問題
\end{list}
\begin{flushright}
の3つのクラスに分けて，豊富にのせてある．また，この本の章の分類に基づいて，
問題集の章末にも，適当な応用問題などをのせておいた．学ぶ人の理解の度合いに
応じて，クラスを選んで，問題を解き，実力を養われることを望んでいる．

なお，この本の問題のなかで，＊印をつけたものは，時間に余裕がある場合に，
説明を加えて教えるのに利用される候補としてあげたものである．

1992年12月
	著　者　しるす
\end{flushright}
